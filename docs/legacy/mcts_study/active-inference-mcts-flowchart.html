Active Inference + Neural MCTS Algorithm (2025 Paper)

Initial State s₀
Continuous state vector (e.g., robot configuration)
Cross-Entropy Method (CEM) Initialization
μ₀ ~ N(0, I), Σ₀ = σ²I
Initialize action distribution parameters
For each CEM iteration k = 1...K
Sample Action Sequences
a⁽ⁱ⁾ ~ N(μₖ, Σₖ), i = 1...N
Generate N candidate action trajectories
Neural Reward Model
R(s,a) = f_reward(s,a)
Predict expected rewards
Free Energy Computation
F = -R - λ·I(s,a)
R: Expected reward
I: Information gain
λ: Epistemic weight
Uncertainty Estimator
σ²(s) = f_uncertainty(s)
Quantify epistemic uncertainty
MCTS Tree Expansion
For each promising action sequence:
  • Expand tree nodes along trajectory
  • Use neural transition model for state prediction
  • Add exploration bonus based on uncertainty
Value Backup with Uncertainty
Q(s,a) ← (1-α)Q(s,a) + α(R + γV(s'))
U(s,a) ← Update uncertainty estimates
Maintain both value and uncertainty estimates
CEM Distribution Update
Select elite set E (top M actions by free energy)
μₖ₊₁ = mean(E)
Σₖ₊₁ = cov(E)
Final Action Selection
a* = argmin_a F(s,a)
Execute action that minimizes expected free energy

Key Components Integration

Neural Network Components
  • Reward Model: Predicts R(s,a) without rollouts
  • Transition Model: Predicts s' = T(s,a)
  • Uncertainty Network: Estimates epistemic uncertainty
Active Inference Principles
  • Free Energy: F = -log P(o|s) + KL[Q||P]
  • Information Gain: Reduction in uncertainty
  • Expected Free Energy: Future uncertainty + goal divergence
Modified MCTS Operations
  • Selection: Guided by CEM proposals instead of UCB
  • Expansion: Uses neural predictions + exploration bonuses
  • Simulation: Replaced by neural value estimates
  • Backup: Updates both Q-values and uncertainty
